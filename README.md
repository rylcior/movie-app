## Getting Started

To get a local copy up and running follow these simple example steps.

### How to run project localy


1. Clone the repo
   ```sh
   git clone https://github.com/rylcior/movie-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run local server
   ```sh
   npm start
   ```
